Overview Map widget

The Overview Map widget adds an overview map to the application. It shows the current spatial extent of the map display as a grey rectangle relative to the entire spatial extent of the base map service. This widget is located in the upper right corner of the application. It can be easily maximized or minimized by clicking the expand arrow.

Configuring the Overview Map widget

Steps:
  1. Hover the mouse over the Overview Map widget and click the small eye icon to show or hide this widget in your application .
  2. Hover the mouse over the Overview Map widget and click the small edit icon

    This opens the configuration window for this widget.

  3. Check or uncheck the Visible check box.

    This sets the overview map as maximized or minimized when the application starts. You can always manually maximize or minimize it as long as the widget is enabled.

  4. Provide values for Min Width, Min Height, Max Width, and Max Height.

    These values define the size of the inner rectangle.

  5. Click OK to close this window.

Using the Overview Map widget

When the application starts, the Overview Map widget is automatically enabled. Click the expansion arrow at the upper right corner of the application to show or hide it.
