Layer List

The Layer List widget displays operational layers on the map and provides basic functionalities, including the ability to turn the layer on and off, zoom to, transparency, move down/move up, open with attribute table, and description.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

showLegend

Boolean. The default is true—Indicates whether to display the legend or not.
