Change URL of ArcGIS API for JavaScript

Open the env.js file, search the phrase of //apiUrl=[POINT_TO_ARCGIS_API_FOR_JAVASCRIPT];, then set the variable apiUrl to the preferred location of ArcGIS API for JavaScript. The variable could point to a privately or publicly hosted version, such as ArcGIS Online JavaScript API.
