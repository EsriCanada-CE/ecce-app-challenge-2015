Open a widget at app start

Set the attribute openAtStart to true when you want to open a widget when the application starts. The default value is false. The attribute only applies for in-panel widgets. 

"openAtStart": true,

If more than one widget in the widgetOnScreen is set to true, the first one opens.

If more than one widget in the widget pool is set to true, it's the controller's responsibility to define how it opens.

Refer to the config-readme.text file under the stemapp folder for details.
