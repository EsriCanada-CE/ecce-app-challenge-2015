PanelManager class

This class is to manage a panel, such as loading a panel, changing a panel state, destroying a panel, and so on.

Example:

require(["jimu/ PanelManager "], function(PanelManager) { 
  var pm = PanelManager.getInstance();
});

Method details:

getInstance()

It’s a static method that returns the singleton object.

Return type: No return.

showPanel (widgetJson/groupJson)

Loads and opens a panel depends on widget/group configuration.

Parameters:

widgetJson/group: the widget/group JSON object configured in app’s config.json.

Return type: Deferred object. When the load is succeeded, the panel object is resolved.

openPanel

Opens a loaded panel. It changes the panel state to open and calls panel onOpen method. If the panel does not load, do nothing.

Parameters:

Panel id or panel instance object.

Return type: No return.

closePanel

Closes a loaded panel. It changes the panel state to closed and calls panel onClose method. If the panel does not load, do nothing.

Parameters:

Panel id or panel instance object.

Return type: No return.

minimizePanel

Minimizes a loaded panel. It changes the panel state to minimized and calls panel onMinimize method. If the panel does not load, do nothing.

Parameters:

Panel id or panel instance object.

Return type: No return.

maximizePanel

Maximizes a loaded panel. It changes the panel state to maximized and calls panel onMaximize method. If the panel does not load, do nothing.

Parameters:

Panel id or panel instance object.

Return type: No return.

normalizePanel

Normalizes a loaded panel. It changes the panel state to normal and calls panel onNormalize method. If the panel does not load, do nothing.

Parameters:

Panel id or panel instance object.

Return type: No return.

getPanelById(panelId)

Returns the panel identified by the panel id. If the panel does not load, returns undefined.

Parameters:

panelId: The panel id.

Return type:Panel object.

destroyPanel(panel)

Destroy the panel.

Parameters:

Panel id or panel instance object.

Return type: No return.

destroyAllPanels

Destroys all of the loaded panels.

Return type: No return.
