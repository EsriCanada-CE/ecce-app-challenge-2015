FeaturelayerServiceBrowser class

The FeaturelayerServiceBrowser makes it easy to filter layers with type 'Feature Layer' from MapService or FeatureService and construct service tree by specified service URL. Only query-enabled service will show.

AMD Module Require:

require(["jimu/dijit/FeaturelayerServiceBrowser"], function(FeaturelayerServiceBrowser) { /* code goes here */ });

Constructor:

new FeaturelayerServiceBrowser (params, srcNodeRef)

Create a new FeaturelayerServiceBrowser dijit.

Parameters:

<Object> params: Required. Parameters for the FeaturelayerServiceBrowser dijit.

<DOMNode | String> srcNodeRef: Optional. HTML element where the print widget button and the drop-down list are rendered.

params properties:

<String> portalUrl: Optional. The url to filter. The url must be a rest service url.

<Boolean> multiples: Optional. Default is false. By default, user can only select one feature layer with radio button. If set to true, user can select multiple feature layers with checkbox.

<String>[] types: Optional. FeaturelayerServiceBrowser supports to filter feature layer by geometry type. The available values are point,polyline and polygon. By default, the value of types is ['point','polyline','polygon'] and dijit doesn't filter feature layer by geometry type.

Example:

require(['jimu/dijit/FeaturelayerServiceBrowser'], function(FeaturelayerServiceBrowser){
					var url = "http://sampleserver1.arcgisonline.com/ArcGIS/rest/services";
					//it also supports following format urls
				    //folder url:  http://sampleserver1.arcgisonline.com/ArcGIS/rest/services/Demographics
				    //service url: http://tryitlive.arcgis.com/arcgis/rest/services/GeneralPurpose/MapServer
				    //group layer url: http://tryitlive.arcgis.com/arcgis/rest/services/GeneralPurpose/MapServer/0
				    //group layer url: http://tryitlive.arcgis.com/arcgis/rest/services/GeneralPurpose/MapServer/1
				    //layer url: http://tryitlive.arcgis.com/arcgis/rest/services/GeneralPurpose/MapServer/2
					var browser = new FeaturelayerServiceBrowser({
						url: url,
						multiple: true,
				  		types: ["point"]
					}, srcNodeRef);
					...
				});

Methods:

setUrl(url)

Use the specified url to filter feature layers.

Paremeters:

<Renderer> url: Required. The rest service url to filter.

getSelectedItems()

Get information of selected layers.

Return type: [{name, url, definition}]
