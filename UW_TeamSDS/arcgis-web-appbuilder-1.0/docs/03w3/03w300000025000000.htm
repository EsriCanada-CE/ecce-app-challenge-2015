Themes

A theme is a template framework representing the look and feel of an application. Contents in a theme include a collection of panels, styles and layouts, and a set of pre-configured theme widgets. One application can include more than one theme, but can only use one theme while running.

Web AppBuilder provides out-of-box themes. The Themes tab supports the following functions:

  • Select a theme—Foldable Theme or Tab Theme.
  • Select a style—The available style vary depending on which theme is selected.
  • Select a layout—The available layouts vary depending on which theme is selected. A layout represents a preset placement of user interface items.
