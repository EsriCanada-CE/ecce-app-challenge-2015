Scalebar widget

The Scalebar widget displays a scalebar on the map. The widget respects various coordinate systems and displays units in English or metric values. When working with Web Mercator or geographic coordinate systems, the scalebar takes into account projection distortion and dynamically adjusts.

Configuring the Scalebar widget

Steps:
  1. Hover the mouse over the Scalebar widget and click the small eye icon to show or hide this widget in your application .
  2. Hover the mouse over the Scalebar widget and click the small edit icon .

    This opens the configuration window for this widget.

  3. Select a unit from the Unit list.
  4. Select a style from the Style list.
  5. Click OK to close this window.

Using the Scalebar widget

When the application starts, the Scalebar widget is automatically enabled and displayed at the lower left corner of the application. The scale value is based on the active spatial reference.
