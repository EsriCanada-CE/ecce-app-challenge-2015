Geocoder

Adds a geographic search box to an application. The widget defaults to the ArcGIS Online World Geocoding Service but is customizable to use one or more ArcGIS Server geocoding services.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

geocoder

An object of ArcGIS API for JavaScript. See the params of Geocoder Constructor.

Example: 

{
  "geocoder": {
    "autoComplete": true,
    "minCharacters": 3,
    "arcgisGeocoder": false,
    "geocoders": [
      {
        "url": "https://geocode.arcgis.com/arcgis/rest/services/World/GeocodeServer",
        "name": "Esri World Geocoder",
        "singleLineFieldName": "SingleLine",
        "placeholder": "Esri World Geocoder"
      },
      {
        "url": "https://gis.lmi.is/arcgis/rest/services/GP_service/geocode_thjonusta_single/GeocodeServer",
        "name": "rrrr",
        "singleLineFieldName": "SingleLineCityName",
        "placeholder": "ffff"
      }
    ]
  }
