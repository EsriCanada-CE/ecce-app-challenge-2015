Bookmark

The Bookmark widget is available for 2D and 3D maps. For 2D maps, it stores a collection of map view extents. For 3D maps, it stores a collection of camera arrays. It also enables end users to add and delete their own bookmarks. Web map bookmarks are automatically used if available.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

bookmarks 2D

object[]; There is no default—The 2D map bookmarks.

  • *name: String; There is no default. The bookmark name.
  • *extent: Object; There is no default. The map extent—The object is the same as the object returned by the Extent.toJson() method in the JavaScript API. See Extent to get more details.
  • *thumbnail: String; Default: images/thumbnail_default.png. As a recommendation, put all images in the images folder.

bookmarks 3D

object[]; There is no default. The 3D map bookmarks.

  • *name: String; There is no default—The bookmark name.
  • *camera: Number[]; There is no default—The camera is an array with a length of 5 or 6. The first fifth of the array is x,y,z, heading, and tilt and the sixth is the WKID. The sixth is optional. If not set, the default WKID is 4326.
  • *thumbnail: String; Default: images/thumbnail_default.png. As a recommendation, put all of the images in the images folder.
