Deploy custom widget and theme

Deploy custom widget

To deploy your custom widget, put it under the Web AppBuilder widget repository, which by default is client\stemapp\widgets. This makes your widget part of the Web AppBuilder. As an example, the UseJQuery widget is an in-panel custom widget that displays in the Choose Widget window when deployed to the widget repository.

If you want to make the custom widget available for a specific app only, copy and paste it in your app's \widgets folder after downloading the app, then edit the app config file to add it manually in the app.

Deploy custom theme

Similar to the custom widget, to deploy the custom theme, put it under the Web AppBuilder theme repository, which by default is client\stemapp\themes . This makes your theme part of the Web AppBuider. As an example, the DemoTheme, as a custom theme, displays in the Themes tab when deployed to the theme repository.

Likewise, if you want to make the custom theme available for a specific app only, copy and paste it in your app's \themes folder after downloading the app, then edit the app's config file to add it manually in the app.
