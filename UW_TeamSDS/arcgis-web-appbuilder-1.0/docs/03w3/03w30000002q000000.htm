Controller

The Controller widget reads and displays application information, such as the title and logo. In addition, it reads widget pool configurations and displays the widget/group icon. Click a specific icon to open or close the corresponding widget and group...

The following table shows the descriptions of the configurable attributes.

Attribute

Description

groupSetting

Object[]. There is no default—The array of the group setting. The groups come from the widget pool. Each setting has two properties: label and type. The label should be the same as the group label in the widget pool. The type has two options: dropDown and openAll. If it’s dropDown, when users click the group icon, a drop-down menu appears. If it’s openAll, when users click the group icon, all widgets in the group open.
