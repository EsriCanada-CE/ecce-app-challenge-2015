My Location

My Location widget locates and zooms to the user’s current location.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

locateButton

An object of ArcGIS API for JavaScript. See the parameters for the LocateButton constructor.

Example: 

{
  "locateButton": {
    "geolocationOptions": {
      "timeout": 10000
    },
    "highlightLocation": true
  }
}
