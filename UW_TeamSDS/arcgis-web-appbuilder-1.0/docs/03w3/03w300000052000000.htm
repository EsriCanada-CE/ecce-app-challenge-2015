About

About widget creates content that displays in the widget.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

*about

Object; default: no default --It has following properties .

  • *aboutContent: String; default: no default —The content displays in About widget.

Example:

{
  "about": {
    "aboutContent": “<div style=\"text-align: center;\"><a href=\"http://www.esri.com/\" target=\"_self\">Hello world!</a><br /></div>"
  }
}
