Home page

Web AppBuilder for ArcGIS (Developer Edition) home page is the place to manage apps. It not only displays information about each app you created but also provides tools to manage it. When log in the portal, you will see all your apps listed on the home page.

Note:

In this release, home page only lists apps created by you as author. It doesn't take into account your role as administator, publisher or user in the portal.

The app information is presented in a tabular view, including App Name, Description, Date modified, and Creator. To edit the app name and description, double-click them.

You can manage apps with a set of tools, including Create New app, Import an app created by Web AppBuilder integrated in ArcGIS Online or Portal for ArcGIS, Edit and Launch the app. In addition, a drop-down menu allows you to Download, Duplicate and Delete the app, as well as Edit App Info and Export as Template.

When you sign in a specified portal to access Web AppBuilder for ArcGIS (Developer Edition), on the upper right corner of the home page, your login information displays.
