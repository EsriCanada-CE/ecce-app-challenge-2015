Map configuration

The following table shows the attributes and descriptions.

Attribute

Description

3D

Boolean. Default: false—Is the map 3D or 2D? If both 3D and 2D are true, it’s an app with the ability to switch between 2D and 3D (not supported in this version).

2D

Boolean. Default: true—Is the map 3D or 2D? If both 3D and 2D are true, it’s an app with the ability to switch between 2D and 3D (not supported in this version).

itemId

string—webmapID or webscene ID.

mapOptions

object—This property is the same as the map API.

position

object. Default—{left:0,Top:0,Bottom:0,Right:0}. The map's position. The viewer uses six properties to position: left, top, bottom, right, width, height, which is the same as the browser.

portalUrl

string--The url where webmap is hosted. If not set, app's portalUrl property is used.

Example:

"map": {
    
    "3D": fale,
    "2D": true,
    "portalUrl": "",
    "itemId": "",
    "position": {
      "left": 0,
      "top": 0,
      "right": 0,
      "bottom": 30,
      "width": 100,
      "height": 100
    },

    "mapOptions": {
      "extent": {
        "xmin": 20, "xmax": 30, "ymin": 40, "ymax": 50, "spatialReference" { "wkid": 4326}
      },
      "center": "",
      "level": 3
    }
  },
