Geoprocessing parameter types

Parameter types used in geoprocessing widges are as follows:

  • GPBoolean
    Check the check box to set the default value to true.
  • GPDouble
    Enter a numeric value for the default or leave the field blank.
  • GPLong
    Enter a long integer value for the default or leave the field blank.
  • GPString
    Enter a text value for the default or leave the field blank.
  • GPLinearUnit
    Enter a numeric value for the default or leave the field blank. Select a default unit of length.
  • GPFeatureRecordSetLayer
    Choose the symbol shown for the input or output features. Reference the Draw widget for the complete set of symbol options.
    Note:
    Currently, Web AppBuilder for ArcGIS does not support attributes for input features.
  • GPRecordSet
  • GPDate
    Enter a  date and time value for the default or leave the fields blank.
  • GPDataFile
    Add a URL in the date field as the default or leave the field blank. 
  • GPRasterDataThis parameter type is currently not supported by Web AppBuilder.
  • GPRasterDataLayer
  • GPMultiValue:GPParameterType
    The interface displayed by the multi-value selector varies based on the parameter type. Click + to add a new default value or - to remove one.
