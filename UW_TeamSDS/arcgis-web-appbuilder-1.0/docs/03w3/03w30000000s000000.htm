Panel configuration

The following table shows the attributes and descriptions.

Attribute

Description

uri

string. Default: BaseWidgetPanel. The panel's main class, derived from BaseWidgetPanel.

position

object—The panel's position, which is the same as the map's position.

positionRelativeTo

string. Default: map—Valid value is map or browser.
