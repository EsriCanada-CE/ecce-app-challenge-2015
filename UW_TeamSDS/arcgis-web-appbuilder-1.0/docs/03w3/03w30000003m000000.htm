Create and import app

Web AppBuilder for ArcGIS (Developer Edition) enables you to create a new app from scratch or import an existing app created by Web AppBuilder integrated in ArcGIS Online or Portal for ArcGIS.

Create new app

Steps:
  1. On the home page, click Create New to create a new app. With this release, only 2D app is available. The option for 3D app is greyed out.
  2. Provide the app name and description, and click OK.
    3. Creating a new app with Web AppBuilder involves the following workflow:
    • Choose the look-and-feel of the app.
    • Specify the web map authored by your portal, which usually contains a basemap and operational layers.
    • Configure the functionalities.
    • Brand the app.
    • Preview the app on different platforms.

Import an app

You can build an app on the top of an existing app by importing it from your ArcGIS Online organization or Portal for ArcGIS.

Steps:
  1. On the home page, click Import to open Import App window.

    It displays available apps created by Web AppBuilder integrated in your portal.

  2. Choose an app, click OK.

    • If the imported app is created by an older version of Web AppBuilder, it will be upgraded auntomatically before imported. When succeeded, you will see the app listed on the home page. Otherwise, a failure message appears and asks you to check the log file

    • If the imported app shares the same version of Web AppBuilder for ArcGIS (Developer Edition), it will be automatically imported and added on the app list.
    3. If an app with the same name already exists, a suffix is automatically added to the imported app name.
