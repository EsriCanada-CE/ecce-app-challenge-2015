Widget properties

The following are additional properties to use in widgets:

Name

Description

id

The unique ID of the widget set in the config.json file or generated by the app container.

icon

The widget’s icon URL.

config

The widget’s config object.

map

The map this widget works with.

appConfig

The app's main config.json file is stored here.

folderUrl

The widget folder’s URL.

state

The widget’s current state, that is, opened or closed.

windowState

The widget’s current window state, that is, normal, maximized, or minimized.

position

The widget's position. It is important for on-screen widget. The position object has 6 properties: left, top, right, bottom, width and height.
