Close widget programmatically

WIth a custom widget, you can make it close programmatically. In this way, the user does not have to click the icon button to close it.

For in-panel widgets, do the following:

define([ 'jimu/PanelManager'], fuction(PanelManager'){
         
         afterSomething: function(){
                   //you can get widgetid by using this.id
                   PanelManager.getInstance().closePanel(widgetid + ‘_panel’);
         }
});

For off-panel widgets, do the following:

WidgetManager. getInstance().closeWidget(‘id’)
Tip:

The closeWidget function just changes the widget state and calls the onClose method. You may override the onClose method for the real close, such as hide or destroy.
