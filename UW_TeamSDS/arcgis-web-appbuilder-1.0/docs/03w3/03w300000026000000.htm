Query

The Query widget enables end users to query information from a map service by executing a predefined query. It works on a single layer.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

queries

Array of query task. Each element has the following child attributes:

  • name: String. There is no default. The name of query task.
  • url: String. There is no default. A feature layer URL for the layer you want to query..
  • filter: Object. There is no default. The expr attribute of filter takes the form of a basic SQL statement and uses it to filter features.
  • Popup: Object. There is no default. Popup includes two attributes:title and fields. Title is a string. The default is the displayFieldName attribute of the layer. Fields is an array of object. Each element of fields includes five attributes:name, alias, type, specialType and showInInfoWindow.
  • resultsSymbol: Object. There is no default. This is the symbol of query results.
  • objectIdField:String. There is no default. This is the objectIdField of layer.

Example: 

{
    "queries": [
        {
            "name": "Cities",
            "url": "http://sampleserver6.arcgisonline.com/arcgis/rest/services/SampleWorldCities/MapServer/0",
            "filter": {
                "logicalOperator": "AND",
                "parts": [
                    {
                        "fieldObj": {
                            "name": "POP_RANK",
                            "label": "POP_RANK",
                            "shortType": "number",
                            "type": "esriFieldTypeInteger"
                        },
                        "operator": "numberOperatorIs",
                        "valueObj": {
                            "isValid": true,
                            "value": 3
                        },
                        "interactiveObj": "",
                        "expr": "POP_RANK = 3"
                    }
                ],
                "expr": "POP_RANK = 3"
            },
            "popup": {
                "title": "${CITY_NAME}",
                "fields": [
                    {
                        "name": "CITY_NAME",
                        "alias": "CITY_NAME",
                        "type": "esriFieldTypeString",
                        "specialType": "none",
                        "showInInfoWindow": false
                    }
                ]
            },
            "resultsSymbol": {
                "color": [
                    0,
                    0,
                    128,
                    128
                ],
                "size": 18,
                "angle": 0,
                "xoffset": 0,
                "yoffset": 0,
                "type": "esriSMS",
                "style": "esriSMSCircle",
                "outline": {
                    "color": [
                        0,
                        0,
                        128,
                        255
                    ],
                    "width": 0.75,
                    "type": "esriSLS",
                    "style": "esriSLSSolid"
                }
            },
            "objectIdField": "OBJECTID"
        }
    ]
}
